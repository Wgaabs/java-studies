# 📦 Composition Order Exercise

Sistema de pedidos em Java que demonstra conceitos fundamentais de Orientação a Objetos: composição, enumerações tipadas e formatação profissional.

## 🛠️ Tecnologias
- Java 17/21
- Git & GitHub
- IntelliJ IDEA

## 📚 Conceitos Aplicados
- ✅ **Enumerações** (`OrderStatus`) para type-safety
- ✅ **Composição** (`Order` → `OrderItem` → `Product`)
- ✅ **Coleções** (`ArrayList`) e loops
- ✅ **Formatação** de data e moeda (`SimpleDateFormat`, `printf` + `Locale.US`)
- ✅ **Encapsulamento** e boas práticas

## ▶️ Como Rodar
1. Abra a pasta `java-course/composition-order-exercise` no IntelliJ
2. Execute `Main.java`
3. Siga os prompts do console
